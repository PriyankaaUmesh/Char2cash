# Char2Cash

Char2Cash is a climate-tech web application designed to turn agricultural waste (crop residues) into climate wealth by preventing crop residue burning. It creates a marketplace connecting farmers, waste collectors, and investors, providing them with interactive dashboards to track activities, manage waste delivery, and monitor carbon offset credits.

## Key Features

- **Farmer Dashboard**: Allows farmers to report available crop residue, schedule collections, and receive cash/credits for avoiding residue burning.
- **Collector Dashboard**: Helps waste collectors manage collection routes, update status of residue pickups, and log delivered quantities.
- **Investor Dashboard**: Enables investors to fund biochar conversion projects, track environmental impact metrics, and monitor carbon credit generation.
- **Interactive UI**: Styled with clean, modern components and tailored layouts for an optimal user experience.

## Project Structure

This project consists of:
- **`quick-launchpad`**: The core React + Vite frontend application built with TypeScript, Tailwind CSS, and Shadcn UI, with Supabase integration for back-end database services.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or bun

### Setup and Running locally

1. Navigate to the frontend directory:
   ```bash
   cd quick-launchpad
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.
